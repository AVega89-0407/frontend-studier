# 📚 Frontend Studier

En personlig referensapp byggd med React + Bootstrap för att samla cheat sheets, kodexempel och kommandon på ett ställe.

## Vad finns i appen?

| Sektion | Innehåll |
|---|---|
| 🅱 Bootstrap | Komponenter med levande preview och kopierbar kod |
| ⟨⟩ HTML | Semantik, formulär, tillgänglighet och SEO-taggar |
| \# CSS | Flexbox, grid, positionering och utility-mönster |
| JS | Variabler, array-metoder, async och vanliga mönster |
| ⚛ React | Hooks, props, villkorlig rendering och vanliga mönster |
| ⎇ Git | Kommandon för grenar, historik och att ångra — klicka för att kopiera |
| ⬡ ASP.NET | Razor-syntax, C#-mönster, Entity Framework och dotnet CLI |

## Kom igång

```bash
npm install
npm run dev
```

## Byggd med

- [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- [Bootstrap 5](https://getbootstrap.com/)
- [React Router](https://reactrouter.com/)
- [Prism React Renderer](https://github.com/FormidableLabs/prism-react-renderer) — syntax highlighting
